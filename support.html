Frequently Asked Questions

[iOS] iCloud Sync isn't working between my devices

SimpleFit Tracker for iOS uses your private iCloud account to sync workouts between Apple devices. To troubleshoot:

  1. Confirm both devices are signed in to the same iCloud account (Settings → [Your Name] → check Apple ID)
  2. iCloud Drive must be enabled (Settings → [Your Name] → iCloud → iCloud Drive → On)
  3. SimpleFit Tracker must be enabled in iCloud (Settings → [Your Name] → iCloud → See All → SimpleFit Tracker → On)
  4. Make sure both devices have an internet connection
  5. Open SimpleFit on both devices and pull-to-refresh on the calendar

Sync usually completes within seconds to a few minutes. If it still doesn't work after 10 minutes, contact us with your iOS version and a screenshot.

Note: SimpleFit Tracker for Android does not currently sync to the cloud. Data stays on your device. You can transfer data between Android devices via the JSON export/import feature (Settings → Data).

Apple Health / Health Connect workouts don't appear in SimpleFit

SimpleFit can import workouts from your platform's health service, but only if you've granted permission:

iOS — Apple Health:

  1. Open the Apple Health app on your iPhone
  2. Tap your profile picture (top right) → Apps and Services → SimpleFit Tracker
  3. Enable all categories: Workouts, Active Energy, Step Count
  4. Return to SimpleFit and trigger an import in Settings → Apple Health

Android — Health Connect:

  1. Open the Health Connect app (pre-installed on Android 14+ or available in the Play Store)
  2. App permissions → SimpleFit Tracker → grant read access to Exercise Sessions
  3. In SimpleFit: Settings → Health Connect → toggle ON → tap Sync now

Note: SimpleFit reads workouts only — no heart-rate samples or location data.

How do I unlock SimpleFit Pro after my trial?

After your 3-day trial, SimpleFit Pro is a one-time purchase that unlocks all premium features forever. To purchase:

  1. Open SimpleFit → Settings → SimpleFit Pro
  2. Tap Unlock Pro
  3. iOS: confirm with Face ID / Touch ID. The price is the localized App Store price for your region.
  4. Android: confirm via Google Play purchase flow. The price is the localized Google Play price for your region.

If you previously purchased on another device signed in to the same Apple ID (iOS) or Google account (Android), tap Restore Purchases to unlock without paying again.

How can I export my data?

Your data belongs to you. SimpleFit lets you export it any time on both platforms:

  1. Open Settings → Data → Export CSV or Export JSON
  2. Choose CSV (for spreadsheets) or JSON (full backup with custom sports + categories)
  3. Share or save the file via the system share sheet (iOS Share Sheet / Android Intent.ACTION_SEND)

On Android you can also re-import a JSON backup via Settings → Data → Import JSON — useful when switching devices.

How do I delete all my data?

iOS:

  • Workouts & settings: Settings → Data & Privacy → Reset App
  • iCloud copies: Settings → [Your Name] → iCloud → Manage Account Storage → SimpleFit Tracker → Delete
  • Apple Health workouts: open the Apple Health app → Browse → Workouts → swipe-delete entries

Android:

  • Workouts & settings: Uninstall the app from your device — all local data is removed.
  • Quick clear (debug): Settings → Apps → SimpleFit Tracker → Storage → Clear Data
  • Health Connect: open Health Connect app → App permissions → SimpleFit Tracker → revoke access. Data in Health Connect itself is not affected.

Both: Right-to-erasure request — email hello@tekneio.com

The widget on my home screen isn't updating

iOS: The system controls widget refresh intervals. Try:

  1. Open SimpleFit and log a workout — this triggers a widget reload
  2. Long-press the widget → Edit Widget and change a setting to force a refresh
  3. Restart your iPhone if the widget shows stale data for more than a day

Android: Android's Glance widgets refresh every ~30 minutes by default. To force an update:

  1. Open SimpleFit and log a workout (triggers worker re-run)
  2. Long-press the widget → remove and re-add it
  3. Check battery optimizations: Settings → Apps → SimpleFit → Battery → unrestricted
Notifications aren't arriving

iOS:

  1. Settings → Notifications → SimpleFit Tracker → Allow Notifications: ON
  2. Inside SimpleFit: Settings → Reminders → enable and pick a time
  3. Make sure Focus mode (Do Not Disturb) is not blocking the app

Android:

  1. Android 13+: when toggling reminders ON, grant the POST_NOTIFICATIONS permission
  2. Settings → Apps → SimpleFit Tracker → Notifications → enabled
  3. Check that battery optimization isn't disabling background work for SimpleFit
  4. Inside SimpleFit: Settings → Reminders → enable + pick time
The app crashed or behaves unexpectedly

We don't collect crash logs automatically (no analytics SDK), so we rely on your report. Please email hello@tekneio.com with:

  • What you were doing when it crashed
  • Platform: iOS or Android
  • Device model and OS version (iOS: Settings → General → About / Android: Settings → About phone)
  • SimpleFit version (Settings → About inside the app)
  • Optional: a screen recording of the issue
Will my data work on a new device?

iOS → iOS: Yes — sign in to the same iCloud account on the new iPhone, install SimpleFit, grant iCloud permission, and your workouts sync down within minutes.

Android → Android: SimpleFit for Android doesn't currently sync to the cloud. To transfer data: on the old device, Settings → Data → Export JSON, save the file (e.g. to Drive or via Share Intent). On the new device, install SimpleFit and use Settings → Data → Import JSON.

Cross-platform (iOS ↔ Android): Use the same JSON export/import process. CSV export works too but does not include custom sports / focus categories.

Is SimpleFit really subscription-free?

Yes. SimpleFit Pro is a one-time purchase, not a subscription. There are no recurring charges, no auto-renewals, no premium tier upsells. Buy once, keep forever — on iOS across all iPhones signed in to the same Apple ID, on Android across all devices signed in to the same Google account.

Privacy & Data

SimpleFit follows a strict data-minimization principle. On iOS, your workouts live in your private iCloud or on your device. On Android, your data stays exclusively on your device. We do not run our own servers, do not use analytics, and do not show ads.

Häufige Fragen

[iOS] iCloud-Sync funktioniert nicht zwischen meinen Geräten

SimpleFit Tracker für iOS synchronisiert Trainings über deine private iCloud zwischen Apple-Geräten. So kannst du das Problem eingrenzen:

  1. Beide Geräte müssen mit derselben Apple ID angemeldet sein (Einstellungen → [Dein Name])
  2. iCloud Drive muss aktiv sein (Einstellungen → [Dein Name] → iCloud → iCloud Drive → Ein)
  3. SimpleFit Tracker in iCloud aktivieren (Einstellungen → [Dein Name] → iCloud → Alle anzeigen → SimpleFit Tracker → Ein)
  4. Beide Geräte brauchen eine Internetverbindung
  5. SimpleFit auf beiden Geräten öffnen und im Kalender nach unten ziehen, um zu aktualisieren

Der Sync ist meist innerhalb weniger Minuten fertig. Wenn nach 10 Minuten nichts passiert, melde dich bei uns — gerne mit iOS-Version und Screenshot.

Hinweis: SimpleFit Tracker für Android synchronisiert derzeit nicht in die Cloud. Daten bleiben auf dem Gerät. Zum Übertragen zwischen Android-Geräten nutze den JSON-Export/Import (Einstellungen → Daten).

Apple-Health / Health-Connect-Trainings erscheinen nicht in SimpleFit

SimpleFit kann Trainings aus dem Gesundheitsdienst deiner Plattform importieren, sobald du den Zugriff erlaubst:

iOS — Apple Health:

  1. Öffne die Apple-Health-App auf deinem iPhone
  2. Tippe oben rechts auf dein Profilbild → Apps und Dienste → SimpleFit Tracker
  3. Aktiviere alle Kategorien: Workouts, Aktive Energie, Schritte
  4. Zurück in SimpleFit → Einstellungen → Apple Health → Import auslösen

Android — Health Connect:

  1. Öffne die Health-Connect-App (auf Android 14+ vorinstalliert oder aus Google Play installieren)
  2. App-Berechtigungen → SimpleFit Tracker → Lese-Zugriff auf Exercise Sessions erlauben
  3. In SimpleFit: Einstellungen → Health Connect → Toggle EIN → Jetzt syncen tippen

Hinweis: SimpleFit liest nur Workouts — keine einzelnen Herzfrequenz-Werte oder Standortdaten.

Wie schalte ich SimpleFit Pro nach dem Trial frei?

Nach den 3 Tagen Trial ist SimpleFit Pro ein einmaliger Kauf, der dauerhaft alle Pro-Features freischaltet. So gehts:

  1. SimpleFit öffnen → Einstellungen → SimpleFit Pro
  2. Auf Pro freischalten tippen
  3. iOS: mit Face ID / Touch ID bestätigen. Der angezeigte Preis ist der lokalisierte App-Store-Preis deiner Region.
  4. Android: über den Google-Play-Kauf-Flow bestätigen. Der Preis ist der lokalisierte Google-Play-Preis deiner Region.

Hast du bereits auf einem anderen Gerät mit derselben Apple ID (iOS) oder Google-Konto (Android) gekauft? Tippe auf Käufe wiederherstellen, um ohne erneute Zahlung freizuschalten.

Wie exportiere ich meine Daten?

Deine Daten gehören dir — du kannst sie jederzeit auf beiden Plattformen exportieren:

  1. Einstellungen → Daten → Als CSV exportieren oder JSON-Backup exportieren
  2. CSV (für Tabellen) oder JSON (volles Backup inkl. eigener Sportarten + Kategorien) wählen
  3. Über das System-Share-Sheet teilen oder speichern (iOS Share Sheet / Android Intent.ACTION_SEND)

Auf Android kannst du ein JSON-Backup auch wieder importieren via Einstellungen → Daten → JSON-Backup importieren — praktisch beim Gerätewechsel.

Wie lösche ich alle meine Daten?

iOS:

  • Trainings & Einstellungen: Einstellungen → Daten & Datenschutz → App zurücksetzen
  • iCloud-Kopien: Einstellungen → [Dein Name] → iCloud → Speicher verwalten → SimpleFit Tracker → Löschen
  • Apple-Health-Trainings: Apple-Health-App → Übersicht → Workouts → einzelne Einträge nach links wischen

Android:

  • Trainings & Einstellungen: App vom Gerät deinstallieren — alle lokalen Daten werden entfernt.
  • Schnell-Clear (Debug): Einstellungen → Apps → SimpleFit Tracker → Speicher → Daten löschen
  • Health Connect: Health-Connect-App → App-Berechtigungen → SimpleFit Tracker → Zugriff entziehen. Daten in Health Connect selbst sind nicht betroffen.

Beide: Recht auf Löschung — E-Mail an hello@tekneio.com

Das Widget auf meinem Home-Bildschirm aktualisiert sich nicht

iOS: Das System steuert, wie oft Widgets neu laden. Versuche:

  1. SimpleFit öffnen und ein Training eintragen — das löst ein Widget-Update aus
  2. Widget lange drücken → Widget bearbeiten und eine Einstellung ändern
  3. iPhone neustarten, falls das Widget länger als einen Tag veraltet ist

Android: Glance-Widgets refreshen standardmässig alle ~30 Minuten. Forcieren:

  1. SimpleFit öffnen und ein Training loggen (triggert Worker)
  2. Widget lange drücken → entfernen und wieder hinzufügen
  3. Akku-Optimierung prüfen: Einstellungen → Apps → SimpleFit → Akku → uneingeschränkt
Erinnerungen kommen nicht an

iOS:

  1. Einstellungen → Mitteilungen → SimpleFit Tracker → Mitteilungen erlauben: Ein
  2. In SimpleFit: Einstellungen → Erinnerungen → aktivieren und Zeit wählen
  3. Fokus-Modus (Nicht stören) prüfen — er kann Mitteilungen blockieren

Android:

  1. Android 13+: beim Aktivieren der Erinnerungen die POST_NOTIFICATIONS-Berechtigung erteilen
  2. Einstellungen → Apps → SimpleFit Tracker → Benachrichtigungen → aktiviert
  3. Akku-Optimierung darf Hintergrundarbeit für SimpleFit nicht deaktivieren
  4. In SimpleFit: Einstellungen → Erinnerungen → aktivieren + Zeit wählen

Cross-Platform (iOS ↔ Android): Gleicher JSON-Export/Import-Prozess. CSV-Export funktioniert auch, beinhaltet aber keine eigenen Sportarten / Fokus-Kategorien.

Ist SimpleFit wirklich abo-frei?

Ja. SimpleFit Pro ist ein einmaliger Kauf, kein Abo. Keine wiederkehrenden Zahlungen, keine Auto-Verlängerung, keine versteckten Tier-Upgrades. Einmal kaufen, für immer behalten — auf iOS auf allen iPhones mit derselben Apple ID, auf Android auf allen Geräten mit demselben Google-Konto.

Datenschutz & Daten

SimpleFit folgt dem Prinzip der Datensparsamkeit. Auf iOS liegen deine Trainings in deiner privaten iCloud oder auf deinem Gerät. Auf Android bleiben deine Daten ausschliesslich auf dem Gerät. Wir betreiben keine eigenen Server, keine Analyse-Tools, keine Werbung.

Questions fréquentes

[iOS] La synchronisation iCloud ne fonctionne pas entre mes appareils

SimpleFit Tracker pour iOS utilise ton iCloud privé pour synchroniser les entraînements entre appareils Apple. Vérifie :

  1. Les deux appareils sont connectés au même identifiant Apple (Réglages → [Ton nom])
  2. iCloud Drive est activé (Réglages → [Ton nom] → iCloud → iCloud Drive → Activé)
  3. SimpleFit Tracker est activé dans iCloud (Réglages → [Ton nom] → iCloud → Tout afficher → SimpleFit Tracker)
  4. Les deux appareils ont une connexion Internet
  5. Ouvre SimpleFit sur les deux appareils et tire vers le bas pour rafraîchir

La synchro se termine généralement en quelques minutes. Si rien ne se passe après 10 minutes, contacte-nous avec une capture d'écran et la version d'iOS.

Note : SimpleFit Tracker pour Android ne se synchronise pas avec le cloud actuellement. Les données restent sur l'appareil. Pour transférer entre appareils Android, utilise l'export/import JSON (Réglages → Données).

Mes entraînements Apple Health / Health Connect n'apparaissent pas dans SimpleFit

SimpleFit peut importer les entraînements depuis le service santé de ta plateforme, à condition d'avoir l'autorisation :

iOS — Apple Health :

  1. Ouvre l'app Santé d'Apple sur ton iPhone
  2. Touche ta photo de profil (en haut à droite) → Apps et services → SimpleFit Tracker
  3. Active toutes les catégories : Entraînements, Énergie active, Pas
  4. Retourne dans SimpleFit → Réglages → Apple Health → Importer

Android — Health Connect :

  1. Ouvre l'app Health Connect (pré-installée sur Android 14+ ou disponible sur Google Play)
  2. Autorisations des apps → SimpleFit Tracker → accorder l'accès en lecture aux Exercise Sessions
  3. Dans SimpleFit : Réglages → Health Connect → activer → toucher Synchroniser maintenant

Note : SimpleFit ne lit que les entraînements — pas les fréquences cardiaques détaillées ni les données de localisation.

Comment débloquer SimpleFit Pro après l'essai ?

Après les 7 jours d'essai, SimpleFit Pro est un achat unique qui débloque toutes les fonctions premium pour toujours :

  1. Ouvre SimpleFit → Réglages → SimpleFit Pro
  2. Touche Débloquer Pro
  3. iOS : confirme avec Face ID / Touch ID. Le prix affiché est celui de ta région App Store.
  4. Android : confirme via le flux d'achat Google Play. Le prix est celui de ta région Google Play.

Tu as déjà acheté sur un autre appareil avec le même identifiant Apple (iOS) ou compte Google (Android) ? Touche Restaurer les achats pour débloquer sans payer à nouveau.

Comment exporter mes données ?

Tes données t'appartiennent — tu peux les exporter à tout moment sur les deux plateformes :

  1. Réglages → Données → Exporter en CSV ou Exporter sauvegarde JSON
  2. Choisis CSV (pour tableurs) ou JSON (sauvegarde complète avec sports/catégories personnalisés)
  3. Partage ou enregistre via la feuille de partage système (iOS Share Sheet / Android Intent.ACTION_SEND)

Sur Android, tu peux aussi réimporter une sauvegarde JSON via Réglages → Données → Importer sauvegarde JSON — utile lors d'un changement d'appareil.

Comment supprimer toutes mes données ?

iOS :

  • Entraînements & réglages : Réglages → Données et confidentialité → Réinitialiser l'app
  • Copies iCloud : Réglages → [Ton nom] → iCloud → Gérer le stockage → SimpleFit Tracker → Supprimer
  • Entraînements Apple Health : ouvre l'app Santé → Parcourir → Entraînements → glisse-supprime les entrées

Android :

  • Entraînements & réglages : Désinstalle l'app — toutes les données locales sont supprimées.
  • Effacement rapide (debug) : Réglages → Apps → SimpleFit Tracker → Stockage → Effacer les données
  • Health Connect : ouvre l'app Health Connect → Autorisations des apps → SimpleFit Tracker → révoquer l'accès. Les données dans Health Connect lui-même ne sont pas affectées.

Les deux : Droit à l'effacement — e-mail à hello@tekneio.com

Le widget sur mon écran d'accueil ne se met pas à jour

iOS : Le système contrôle la fréquence de rafraîchissement :

  1. Ouvre SimpleFit et enregistre un entraînement — cela force un rafraîchissement
  2. Appui long sur le widget → Modifier le widget et change un réglage
  3. Redémarre ton iPhone si le widget reste figé plus d'une journée

Android : Les widgets Glance se rafraîchissent par défaut toutes les ~30 minutes. Pour forcer :

  1. Ouvre SimpleFit et enregistre un entraînement (déclenche le worker)
  2. Appui long sur le widget → supprime et réajoute-le
  3. Vérifie l'optimisation batterie : Réglages → Apps → SimpleFit → Batterie → non restreinte
Les notifications n'arrivent pas

iOS :

  1. Réglages → Notifications → SimpleFit Tracker → Autoriser : activé
  2. Dans SimpleFit : Réglages → Rappels → activer et choisir une heure
  3. Vérifie le mode Concentration (Ne pas déranger) — il peut bloquer les apps

Android :

  1. Android 13+ : lors de l'activation des rappels, accorde la permission POST_NOTIFICATIONS
  2. Réglages → Apps → SimpleFit Tracker → Notifications → activées
  3. L'optimisation batterie ne doit pas désactiver le travail en arrière-plan
  4. Dans SimpleFit : Réglages → Rappels → activer + choisir l'heure
